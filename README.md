## 运行方法
运行前确保根目录下有.env配置文件，并完整填写好配置信息<br>

如果下载代码运行确保电脑上安装了go环境，然后用命令行切换到项目根目录下执行 go run main.go<br>

或者下载releases中打包好的二进制文件，直接运行就行了。
